# Tangram Puzzle Game

A simple Tangram puzzle game made in Unity.

## How to Play
- Drag the colored pieces with the mouse
- Press **R** to rotate the selected piece
- Place each piece into the black shapes
- Pieces can be moved and rotated until they fit correctly
- When all pieces are placed correctly, you win

## Controls
- Mouse – Drag pieces  
- R – Rotate selected piece  

## Features
- Drag and drop pieces
- Rotation support
- Snapping to target shapes
- Correct placement validation
- Win condition detection
- Sound effects
- Instruction and win UI

## Setup
1. Open the project in Unity
2. Open the main scene
3. Press Play

## Audio
- Click – pick up / rotate piece
- Bump – snap piece
- Pop – puzzle completed
